## 문제 1: 교차 수열 최소 변환

길이 N의 수열 A가 주어졌을 때, 최소한의 원소 교체로 특정 패턴을 만족하는 수열을 만들어야 합니다.

### 조건

수열의 인덱스는 1-based 기준입니다.
- 짝수 인덱스 (i % 2 == 0) → 오름차순(증가하는 방향)
- 홀수 인덱스 (i % 2 == 1) → 내림차순(감소하는 방향)
- 원소 변경 시, 값은 임의의 정수로 교체할 수 있습니다.

### 목표

- 조건을 만족하도록 최소 개수의 원소를 바꿔야 합니다.
- 결과적으로 바꿔야 하는 원소 수를 출력합니다.

### 입력

```
N
A[1] A[2] ... A[N]
```
- 1 ≤ N ≤ 100,000
- 1 ≤ A[i] ≤ 1,000,000,000

### 출력

- 조건을 만족시키기 위해 변경해야 하는 최소 원소 수
