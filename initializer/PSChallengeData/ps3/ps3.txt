## 문제 3: 스마트 도시 내 무인 배달 최적 경로

스마트 도시에는 N개의 교차로와 M개의 양방향 도로가 존재합니다. 각 로봇은 도로와 주문의 시간 제약을 고려하여 최적 경로를 찾아야 합니다.

### 도로 정보

각 도로는 두 교차로 U, V, 도로 거리, 그리고 여러 개의 도로 이용 가능 시간 구간 정보를 가집니다.
- 규칙: 로봇은 도로에 진입할 때, 해당 도로가 허용된 시간 구간 안이어야 합니다.
- 이동 시간: `ceil(거리 / 속도)` 로 계산됩니다.

### 로봇 정보

각 로봇은 이동 속도와 최대 배터리 시간(임무 시작부터 종료까지 사용 가능한 총 시간 상한) 속성을 가집니다.

### 주문 정보

각 주문은 주문 목적지 교차로와, 여러 시간 구간 `[s, e]` 및 해당 구간의 처리 대기 시간 `w` 정보를 가집니다.
- 주문 처리 규칙:
    - 주문 도착 시간이 특정 구간 `[s, e]`에 포함되면, 완료 시각은 `도착시각 + w`가 됩니다.
    - 구간 사이의 빈틈에 도착하면 즉시 처리됩니다 (대기 시간 0).
    - `e = 0`인 구간은 무제한 구간(∞)으로 해석합니다.

### 목표

- 모든 주문을 배달할 수 있는 최소 시간 T를 구합니다.
- 단, 각 로봇은 한 주문만 맡을 수 있고, 배터리 제한 시간 이내에 해당 주문을 완료해야 합니다.
- 불가능하다면 -1을 출력합니다.

### 입력

```
N M R O
도로 정보 M줄: U V 거리 구간수 [시작 종료]...
로봇 정보 R줄: 속도 배터리
주문 정보 O줄: 목적지교차로 구간수 [시작 종료 대기시간]...
```

### 출력

- 모든 주문이 완료될 수 있는 최소 시간 T
- 불가능하면 -1
